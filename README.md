# Projeto-MarketView

Projeto que mostra em uma interface, a cotação das 10 ações e criptomoedas com maior volume de mercado.
Como sou uma pessoa que gosta de observar o mercado financeiro, optei por criar este projeto, para que com um clique eu tenha todas essas informações.

### Bibliotecas Utilizadas

* [CustomTkinter](https://customtkinter.tomschimansky.com/) (Criação de Interface)
* [Pillow](https://pypi.org/project/pillow/) (Utilização de imagens)
* [yfinance](https://pypi.org/project/yfinance/)
* [Selenium](https://selenium-python.readthedocs.io/)
* Requests
* Os
* Pathlib


### Como rodar o projeto

Projeto em desenvolvimento, porém seguindo as propostas, você irá executar o arquivo main.exe, que iniciará o sistema e abrirá uma interface gráfica, lhe mostrando inicialmente as ações e na parte superior você pode alterar para as criptomoedas. (Estou verificando se aumento a quantidade de ativos por aba)
